This project is a Musical Instruments Sales Dashboard developed using Power BI. The dashboard analyzes sales data of different musical instruments such as guitars, pianos, drums, and violins. It provides insights into revenue, sales trends, customer preferences, and purchase modes through interactive charts and visualizations. Power Query is used for data cleaning, and DAX measures are created for calculations like total revenue and total sales. The dashboard helps users understand business performance in a simple and visually effective manner.
